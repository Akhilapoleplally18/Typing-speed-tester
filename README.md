**Typing Speed Tester**
A modern, responsive web application for measuring typing speed and accuracy. This project includes two dedicated modes:
--Alphabets Typing Test (practice typing the alphabet in different sequences)
--Paragraphs Typing Test (type real-world sentences and challenging paragraphs with selectable difficulty)

**Features**
--Animated gradient background and visually attractive design
--Responsive layout for both desktop and mobile devices
--Real-time display of Words Per Minute (WPM) and typing accuracy
--Easy, Medium, and Hard difficulty levels for paragraph mode
--Font Awesome icons and Google Fonts for a professional look
--Separate sections and independent timers for alphabets and paragraphs

**How to Use**
--Open the application in your browser.
--Click “Start Test” on either the Alphabets or Paragraphs section.
--Type the shown text as quickly and accurately as possible.
--Results update live and the test ends after 60 seconds or upon completion.
